## Descripción del proyecto
Este proyecto consiste en el desarrollo de un invernadero controlado con ESP32 en modelo a escala.

La finalidad es representar de forma pequeña el funcionamiento básico de un sistema automatizado, utilizando la ESP32 como unidad de control.

El proyecto busca integrar electrónica, programación y una estructura física a escala para simular el comportamiento de un invernadero controlado.
